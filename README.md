<h2>
Instalação
</h2>
<h3>no terminal :</h3>

**clone o repositorio**
```bash
 git clone https://github.com/Jose-GuilhermeG/Eclipse.git
```

**agora crie um ambiente virtual e instale as derramentas usadas**

no terminal python : 
```bash
python -m venv venv 
```
**depois ative o ambiente**  
```bash
venv\Scripts\activate 
```
**e por fim baixe as ferramentas**
```bash
pip install -r requirements.txt 
```


<h2>
Rodando o codigo
</h2>

entre na pasta do projeto 'Eclipse'
e no terminal coloque:
```bash
python -m manage runserver
```

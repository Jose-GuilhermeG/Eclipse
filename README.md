 <h1 align = 'center'>
    Eclipse
 </h1>
 <p>
    Um trabalho escolar, a eclipse é uma simples loja com foco em produtos de informática.
 </p>
 <h2>
    estrutura
</h2>
    
<p>
    Sua base front é feita em html,css e javascript
    já sua base back é usando <a href = 'https://flask.palletsprojects.com/en/stable/'> flask </a> e seu banco de dados <a href = 'https://www.sqlite.org/'> sqlite </a>
</p>

<h2>
   autenticação
</h2>
<p>
A autenticação é feita usando a biblioteca <a href = 'https://flask-jwt-extended.readthedocs.io/en/stable/'>jwt para flask </a>
</p>

<h1>
como usar
</h1>

<h2>
Pré-requisitos
</h2>
<ul>
   <li>
      Python
   </li>
   <li>
      sqlite3
   </li>
</ul>

<h2>
Instalação
</h2>
<p>
clone o repositorio
<br>
no terminal : git clone https://github.com/Jose-GuilhermeG/Eclipse.git
<br>
agora crie um ambiente virtual e instale as derramentas usadas
<br>
no terminal python : <strong> python -m venv venv </strong>
depois ative o ambiente  : <strong> venv\Scripts\activate </strong>
e por fim baixe as ferramentas : <strong> pip install -r requirements.txt </strong>
</p>

<h2>
Rodando o codigo
</h2>
<p>
Ha um arquivo chamado run, basta executa-lo ou usar : 'python -m flask run' no terminal
</p>
